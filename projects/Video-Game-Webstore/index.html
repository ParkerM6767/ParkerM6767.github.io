Green Lion Gaming - Video Game Webstore

Description:

Green Lion Gaming was a reactive video game webstore made with a tech stack of base html, javascript, with a server-side component of node.js with express, and mySQL Server.

The website contained:

  • Registration/Sign-up page
  • Basic user authentication
  • Shopping cart system; and
  • A video game store page stored in a relational database

Skills I learned:

  • HTML, CSS, JavaScript, Node.js, Express.js, mySQL
  • Team management using Gantt charts and effective collaboration
