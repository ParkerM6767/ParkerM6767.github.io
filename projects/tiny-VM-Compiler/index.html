PL/0 Tiny VM & Compiler

Description:

PL/0 Tiny VM & Compiler is a simple compiler and virtual machine for the PL/0 programming language and grammar. Using the C programming language,

I developed both a compiler that translates PL/0 source code into object code and executable files that could be run on a virtual machine I made that executes this machine code.

Skills I learned:

Through building this project, I learned about compiler design, virtual machine architecture, and low-level programming concepts. Working on this project enhanced my understanding of how high-level programming languages are translated into machine-executable code.
